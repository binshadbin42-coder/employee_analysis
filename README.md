# Employee Data Analysis Project

 Overview
This project analyzes employee data from ABC company to understand employee distribution across teams, salary patterns, and age-related trends.

Steps Performed

- Data Cleaning (Height column corrected with random values between 150–180)
- Team Distribution Analysis
- Position-wise Employee Segmentation
- Age Group Identification
- Salary Expenditure Analysis
- Correlation Analysis between Age & Salary

 Tools Used
- Python
- Pandas
- Matplotlib


Key Insights

- Most employees are in **Sales team**
- Most common position is **Executive**
- Dominant age group is **25–30**
- Highest salary expenditure is in **IT team (Manager position)**
- Age and salary have a **positive correlation**


Files
- analysis.ipynb (Jupyter Notebook with full code)
- data.csv (Dataset used for analysis)

Conclusion

The analysis reveals that employee distribution is higher in the Sales department, with Executives forming the majority role. Most employees fall within the 25–30 age group. Salary expenditure is highest in managerial roles within the IT team. A positive correlation between age and salary indicates that experience tends to increase earnings.
